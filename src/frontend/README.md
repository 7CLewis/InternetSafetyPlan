# Internet Safety Plan
Internet Safety Plan is a web and mobile application that helps parents keep track of all of the internet-connected devices in their home and give them goals to work towards to secure their devices and protect their children online.

## Installation and Setup
To install and run the application, follow these steps:

1. Clone the repository to your local machine: `git clone https://InternetSafety@dev.azure.com/InternetSafety/Plan/_git/internet-safety-plan-ui`
2. Navigate to the project directory: `cd internet-safety-plan`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

The application should now be running at http://localhost:3000.
